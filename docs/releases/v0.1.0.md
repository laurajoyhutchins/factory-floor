# Factory Floor v0.1.0

Factory Floor v0.1.0 closes Milestone 1: a durable reactive graph runtime that executes TypeScript and Python components while keeping runtime truth, artifacts, retries, recovery, and operator evidence explicit and inspectable.

## What shipped

- PostgreSQL-authoritative commands, events, deliveries, executions, attempts, topology, policy decisions, resources, and projection checkpoints.
- Immutable artifact staging and publication through filesystem and S3-compatible adapters, with provenance, reconciliation, and tombstoning.
- A versioned worker HTTP protocol plus TypeScript and Python clients, runners, and deterministic demo components.
- Atomic result publication, lifecycle and lease fencing, durable retry, dead-lettering, cancellation, and startup recovery.
- A complete investigation vertical slice with fan-out, fan-in, a deliberate first-attempt failure, successful replacement, trace reconstruction, and duplicate-free completion.
- Operator inspection APIs and `ff` CLI views for runtime state, traces, artifacts, lineage, resources, policies, and projection status.
